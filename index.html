Filtro para Identificar Perda de Sinal

Para monitorar a perda de sinal de forma eficiente, configure os seguintes filtros:

  • Filtro - Data com.bordo: Insira valores superiores à data e hora desejada.
    Exemplo: > 10/12/2024 02:00
  • Filtro - Proximidade: Configure para não conter veículos próximos às garagens.
    Exemplo: ! *dalastra*
  • Filtro - Observação: Inclua categorias específicas como FROTA, AGREGADO, ou PVZ para refinar os resultados.

Procedimentos Importantes

  • Na Omnilink: Enviar comando de deslacre de motor para veículos em reinicio de pernoite.
  • Rastreadores secundários: Utilize ucv (Consulte o login da liderança na Autotrac) ou 3S.
  • Para veículos do projeto Caminhoneiro: mensagens de macros, se parecem com mensagens de texto livre, devem ser consideradas.

Check List

Siga os passos abaixo para realizar o check list de forma adequada:

  1. Confirmar se há um pedido de check list.
  2. Na Autotrac:
    • Ativar TESTE de botão de pânico.
    • Confirmar com o motorista se o veículo é contêiner ou baú.
    • Enviar o comando de início de check list com ou sem baú conforme a resposta (geralmente é contêiner, ou seja, sem baú).
  3. Pedir mensagem de teste.
  4. Oriente o motorista sobre o procedimento de responder a contra senha do VIAGEM SEGUE NORMAL
  5. Enviar, opcionalmente, os comandos:
    • Proibir engate.
    • Proibir porta carona.
  6. Verificar violações no histórico (últimos comandos e alertas de OBC).
  7. Solicitar tabela de status e verificar se ela veio vazia.
  8. Após as verificações:
    • Enviar fim de check list s/baú ou com, dependendo.
    • Liberar veículo com sirene.
  9. Na aba de Trafegus, atender a solicitação de check list.
  10. Verificar no terminal os sensores gerados e instalados.
